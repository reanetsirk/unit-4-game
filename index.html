Read The Instructions Here:


  1. At the start of the game the user will be given a random number between 19-120.
  2. User will see four crystals. Use the crystals to match your points to the random number
    • Blue
    • Green
    • Red
    • Yellow
  3. Next, user should clink on one of the four crystals. By clicking on a crystal user will be able to add a specific number of points to the users total score.

  4. In order to win the game the user must match their total score to the random number. User will lose the game if the random number is greater than the users score

  5. Each crystal value is hidden until the user clicks on a specific color crystal(Blue,Green,Red,Yellow)
  6. Restating the game will will randomize the crystals.
